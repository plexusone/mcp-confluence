## [v0.1.0] - 2025-12-29

### Highlights

- Initial release with MCP tools for safe Confluence page editing
- Structured content blocks for creating and reading pages
- Lossless XHTML round-trip editing for complex pages

### Added

- `confluence_read_page` tool for reading pages as structured content blocks
- `confluence_read_page_xhtml` tool for reading pages as raw Storage Format XHTML
- `confluence_update_page` tool for updating pages with structured content blocks
- `confluence_update_page_xhtml` tool for updating pages with raw XHTML
- `confluence_create_page` tool for creating new pages with structured blocks
- `confluence_create_table` tool for creating table blocks from structured data
- `confluence_delete_page` tool for deleting pages by ID
- `confluence_search_pages` tool for searching pages using CQL
- Structured block types: Paragraph, Heading, Table, BulletList, NumberedList, Macro, CodeBlock, HorizontalRule
- Storage package with Parse, Render, and Validate functions
- HTML entity support for Storage Format XHTML
- Confluence REST API client with basic authentication
